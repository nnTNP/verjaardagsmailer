Azure App instellen
Eenmalige setup — ca. 10 minuten
Waarom Azure?

Om conceptmails aan te maken in uw Outlook heeft deze app toestemming nodig van Microsoft. Dit gaat via een gratis Azure App Registration. U registreert de app eenmalig, daarna werkt alles automatisch.

U heeft een Microsoft 365 / Azure account nodig. Ga naar portal.azure.com en volg de stappen hieronder.
Stap voor stap
1
Ga naar Azure Portal → App registrations
Open portal.azure.com → App registrations en klik op + New registration.
2
App registreren
Vul in:
• Name: AdminPulse Verjaardagsmailer
• Supported account types: Accounts in this organizational directory only
• Redirect URI: kies Single-page application (SPA) en vul in:
http://localhost
Klik Register.
3
Client ID en Tenant ID kopiëren
Na registratie ziet u het overzicht. Kopieer:
• Application (client) ID
• Directory (tenant) ID
Plak deze hieronder en op het Configuratie-tabblad.
4
API permissions instellen
Ga naar API permissions → Add a permission → Microsoft Graph → Delegated permissions.
Voeg toe:
• Mail.ReadWrite — concepten aanmaken in Outlook
• User.Read — inloggen
Klik daarna op Grant admin consent.
5
Redirect URI controleren
Ga naar Authentication. Controleer dat onder Single-page application de URI staat:
http://localhost
Sla op. De setup is klaar!
Vul hieronder uw gegevens in ter controle, dan neemt de app ze automatisch over op het Configuratie-tabblad.
